Concept Final Project

Vacuum Forming Machine

For my final project i make a Vacuum Forming Machine.
Two years ago i building a vaccuum forming machine but with some problems.

The problems are:
-Positioning the material in the vacuum table;
-Release of gases
- Do not know the temperature to mold the material

After this problems i resolve make another type of vacuum forming machine.
This time i'm going to build a vacuum forming user frindly.
In this machine the user selects the program intended for type of material will use, after the user select the program the machine heat the material and automatically knows that material is prepared to be molded and again automatically the machine put the material in exact position on vacuum table.

This machine have sensor to knows if the material is prepared and if have the exact temmperature for the table vacuum. The frame where you put the material is in an axis with a motor, this axis make the material to go down from the heating to the vacuum.
For the gases i decide to make a closed box with a door to put a meterial in the machine and an air extraction point for the gases to be extracted to the street.
